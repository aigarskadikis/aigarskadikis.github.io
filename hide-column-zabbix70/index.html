Hide column in Zabbix 7.0 GUI

What is this?

Instruction to run CSS overrides for Zabbix 7.0.

Installations on frontend server

cd /usr/share/zabbix/modules
wget https://github.com/gr8b/zabbix-module-uitwix/archive/refs/heads/main.zip
ls -lh
unzip main.zip
rm main.zip

Enable at application layer

Open "Administration" => "General" => "Modules" => click "Scan directory", make sure "UITwix" is Enabled


Options of CSS rules

TargetCSS
 
regex: /zabbix.php.action=host.list/
/* Data Collection => hosts => hide "Templates" */
.list-table thead tr th:nth-child(10),.list-table tbody tr td:nth-child(10) {display:none}
regex: /zabbix.php.*action=item.list.*context=host/
/* Data Collection => hosts => items => hide "Tags" */
.list-table thead tr th:nth-child(11),.list-table tbody tr td:nth-child(11) {display:none}
regex: /.*/
/* move modal win to very top */
.overlay-dialogue.modal-popup {top:0}
regex: /.*/
/* remove gap at bottom */
.overlay-dialogue.modal .overlay-dialogue-body {max-height: calc(100vh - 96px); margin:0}
regex: /.*/
/* extend width for trigger expression */
#expression, #description, #opdata, #event_name, #name, #url_name, #url, #snmp_oid, #key, #delay, #recovery_expression, params_f {width:90% !important;}

Database operations to backup, restore, distribute settings

Which users are using UI Twix and what are the CSS rules. Backup settings:
SELECT userid,value_str FROM profiles WHERE idx='uitwix-css';
Overwrite settings (settings per some user must exist before running. otherwise - nothing to overwrite) for all users:
UPDATE profiles SET value_str='[{"action":"regex: \/zabbix.php.action=host.list\/","css":"\/* Data Collection => hosts => hide \"Templates\" *\/\r\n.list-table thead tr th:nth-child(10),.list-table tbody tr td:nth-child(10) {display:none}\r\n"},{"action":"regex: \/zabbix.php.*action=item.list.*context=host\/","css":"\/* Data Collection => hosts => items => hide \"Tags\" *\/\r\n.list-table thead tr th:nth-child(11),.list-table tbody tr td:nth-child(11) {display:none}"},{"action":"regex: \/.*\/","css":"\/* move modal win to very top *\/\r\n.overlay-dialogue.modal-popup {top:0}"},{"action":"regex: \/.*\/","css":"\/* remove gap at bottom *\/\r\n.overlay-dialogue.modal .overlay-dialogue-body {max-height: calc(100vh - 96px); margin:0}\r\n"},{"action":"regex: \/.*\/","css":"\/* extend width for trigger expression *\/\r\n#expression, #description, #opdata, #event_name, #name, #url_name, #url, #snmp_oid, #key, #delay, #recovery_expression, params_f {width:90% !important;}\r\n"}]' WHERE idx='uitwix-css';
Overwrite settings (settings per user object must exist before running. otherwise - nothing to overwrite) for single user:
UPDATE profiles SET value_str='[{"action":"regex: \/zabbix.php.action=host.list\/","css":"\/* Data Collection => hosts => hide \"Templates\" *\/\r\n.list-table thead tr th:nth-child(10),.list-table tbody tr td:nth-child(10) {display:none}\r\n"},{"action":"regex: \/zabbix.php.*action=item.list.*context=host\/","css":"\/* Data Collection => hosts => items => hide \"Tags\" *\/\r\n.list-table thead tr th:nth-child(11),.list-table tbody tr td:nth-child(11) {display:none}"},{"action":"regex: \/.*\/","css":"\/* move modal win to very top *\/\r\n.overlay-dialogue.modal-popup {top:0}"},{"action":"regex: \/.*\/","css":"\/* remove gap at bottom *\/\r\n.overlay-dialogue.modal .overlay-dialogue-body {max-height: calc(100vh - 96px); margin:0}\r\n"},{"action":"regex: \/.*\/","css":"\/* extend width for trigger expression *\/\r\n#expression, #description, #opdata, #event_name, #name, #url_name, #url, #snmp_oid, #key, #delay, #recovery_expression, params_f {width:90% !important;}\r\n"}]' WHERE idx='uitwix-css' AND userid=3;
Uninstall all settings related to module:
DELETE FROM profiles WHERE idx='uitwix-css';
