Zabbix proxy 7.0 on Raspberry Pi 4

What is this?

Instruction to setup Zabbix Proxy on raspberry Pi.

Pros

  • Setup OS layer where no writes are done to SD card. Prolong life of SD card
  • Disk will run of a speed of LPDDR4
  • Can use Pi to test an installation of different software. Revert back to it's original state with a single system reboot

Cons

  • Timeconsuming to update/patch OS. Every time need to patch the whole system, this will cost at least 2 restarts
  • OS will not remember installed packages
  • Old logs will be missing in case of a system restart

Prerequsites

Raspberry Pi 4 (4GB or 8GB model)

Download Raspberry Pi OS Lite image

This is tested with:


Disk image:
https://downloads.raspberrypi.com/raspios_lite_arm64/images/raspios_lite_arm64-2024-11-19/2024-11-19-raspios-bookworm-arm64-lite.img.xz

Install command:
echo "Download Zabbix 7.0 LTS repository" && \
wget "https://repo.zabbix.com/zabbix/7.0/raspbian/pool/main/z/zabbix-release/zabbix-release_latest_7.0+debian12_all.deb" && \

echo "Install Zabbix 7.0 LTS repository" && \
sudo dpkg -i zabbix-release_latest_7.0+debian12_all.deb && \

echo "Update system repositories" && \
sudo apt update && \

echo "Install Zabbix proxy, Zabbix agent and Zabbix utilities" && \
sudo apt install zabbix-proxy-sqlite3 zabbix-get zabbix-sender zabbix-js zabbix-agent vim -y && \

echo "Overwrite Zabbix proxy configuration file" && \
echo "Include=/boot/firmware/zabbix_proxy.d/*.conf" | sudo tee /etc/zabbix/zabbix_proxy.conf && \

echo "Remove zabbix_proxy.d directory to not bother" && \
sudo rm /etc/zabbix/zabbix_proxy.d -rf && \

echo "Create zabbix_proxy.conf in the fat32 partition. usefull for SD cloning and Windows administrators" && \
sudo mkdir -p /boot/firmware/zabbix_proxy.d && \

echo "Install proxy configuration file. For all options consult:" && \
echo "https://git.zabbix.com/projects/ZBX/repos/zabbix/browse/conf/zabbix_proxy.conf?at=refs%2Fheads%2Frelease%2F7.0" && \
echo "
AllowRoot=1
CacheSize=512M
DBName=/tmp/zabbix_proxy.sqlite3
EnableRemoteCommands=1
Fping6Location=/usr/bin/fping6
FpingLocation=/usr/bin/fping
LogFileSize=100
LogFile=/var/log/zabbix/zabbix_proxy.log
LogRemoteCommands=1
PidFile=/run/zabbix/zabbix_proxy.pid
ProxyBufferMode=disk
Server=zabbixServer
SocketDir=/run/zabbix
StatsAllowedIP=127.0.0.1
" | sudo tee /boot/firmware/zabbix_proxy.d/zabbix_proxy.conf && \

echo "Overwrite Zabbix agent configuration file" && \
echo "Include=/boot/firmware/zabbix_agentd.d/*.conf" | sudo tee /etc/zabbix/zabbix_agentd.conf && \

sudo rm /etc/zabbix/zabbix_agentd.d -rf && \
sudo mkdir -p /boot/firmware/zabbix_agentd.d && \

echo "Install agent configuration file. For all options consult:" && \
echo "https://git.zabbix.com/projects/ZBX/repos/zabbix/browse/conf/zabbix_agentd.conf?at=release%2F7.0" && \
echo "
AllowKey=system.run[*]
AllowRoot=1
BufferSize=65535
LogFileSize=100
LogFile=/var/log/zabbix/zabbix_agentd.log
LogRemoteCommands=1
PidFile=/run/zabbix/zabbix_agentd.pid
Server=127.0.0.1
ServerActive=127.0.0.1
" | sudo tee /boot/firmware/zabbix_agentd.d/zabbix_agentd.conf && \

echo "Create a directory to redefine systemd definition for agent" && \
sudo mkdir -p /etc/systemd/system/zabbix-agent.service.d && \

echo "Set Zabbix agent ro run under user root" && \
echo "
[Service]
User=root
Group=root
" | sudo tee /etc/systemd/system/zabbix-agent.service.d/override.conf && \




sudo mkdir -p /boot/firmware/zabbix_agent2.d && \

echo "Install agent 2 configuration file. For all options consult:" && \
echo "https://git.zabbix.com/projects/ZBX/repos/zabbix/browse/conf/zabbix_agent2.conf?at=release%2F7.0" && \
echo "
AllowKey=system.run[*]
BufferSize=65535
LogFileSize=100
LogFile=/var/log/zabbix/zabbix_agent2.log
LogRemoteCommands=1
PidFile=/run/zabbix/zabbix_agent2.pid
PluginSocket=/run/zabbix/agent.plugin.sock
ControlSocket=/run/zabbix/agent.sock
ListenPort=12050
Server=127.0.0.1
ServerActive=127.0.0.1
" | sudo tee /boot/firmware/zabbix_agent2.d/zabbix_agent2.conf && \

echo "Create a directory to redefine systemd definition for agent 2" && \
sudo mkdir -p /etc/systemd/system/zabbix-agent2.service.d && \

echo "Set Zabbix agent ro run under user root" && \
echo "
[Service]
User=root
Group=root
" | sudo tee /etc/systemd/system/zabbix-agent2.service.d/override.conf && \






echo "Create a directory to redefine systemd definition for proxy" && \
sudo mkdir -p /etc/systemd/system/zabbix-proxy.service.d && \

echo "Set Zabbix proxy ro run under user root" && \
echo "
[Service]
User=root
Group=root
" | sudo tee /etc/systemd/system/zabbix-proxy.service.d/override.conf && \

echo "Reload systemd service definitions" && \
sudo systemctl daemon-reload && \

echo "Make sure proxy and agent is down" && \
sudo systemctl stop zabbix-proxy zabbix-agent && \

echo "Remove logs" && \
sudo rm -rf /var/log/zabbix/* /tmp/zabbix* && \

echo "Enable Zabbix services to boot at startup" && \
sudo systemctl enable zabbix-proxy zabbix-agent && \

echo "Disable wifi management" && \
sudo systemctl disable --now wpa_supplicant && \

echo "Upgrade whole operating system" && \
sudo apt upgrade -y  && \

echo "Install script to restart proxy if conf changed" && \
echo '
#!/bin/bash
A=/boot/firmware/.proxy
B=`md5sum /boot/firmware/zabbix_proxy.d/*|md5sum|grep -Eo "^\S+"`
cat "$A"|grep "$B"||systemctl restart zabbix-proxy && echo -n "$B">"$A"
' | sudo tee /usr/local/bin/proxy.d.sh && \
sudo touch /boot/firmware/.proxy && \
sudo chmod +x /usr/local/bin/proxy.d.sh && \
echo "18 * * * * root /usr/local/bin/proxy.d.sh" | sudo tee /etc/cron.d/trigger_zabbix_proxy_restart && \

echo "Install script to restart Zabbix agent if conf changed" && \
echo '
#!/bin/bash
A=/boot/firmware/.agentd
B=`md5sum /boot/firmware/zabbix_agentd.d/*|md5sum|grep -Eo "^\S+"`
cat "$A"|grep "$B"||systemctl restart zabbix-agent && echo -n "$B">"$A"
' | sudo tee /usr/local/bin/agentd.d.sh  && \
sudo touch /boot/firmware/.agentd && \
sudo chmod +x /usr/local/bin/agentd.d.sh && \
echo "28 * * * * root /usr/local/bin/agentd.d.sh" | sudo tee /etc/cron.d/trigger_zabbix_agentd_restart && \

echo "Disable wifi and bluetooth" && \
sudo grep "dtoverlay=disable-wifi" /boot/firmware/config.txt || echo "dtoverlay=disable-wifi" | sudo tee --append /boot/firmware/config.txt
sudo grep "dtoverlay=disable-bt" /boot/firmware/config.txt || echo "dtoverlay=disable-bt" | sudo tee --append /boot/firmware/config.txt
Move agent 2 plugins:
sudo mv /etc/zabbix/zabbix_agent2.d/plugins.d /boot/firmware/zabbix_agent2.d
rm -rf /etc/zabbix/zabbix_agent2.d
Set hostname and enable overlay filesystem: 
sudo raspi-config
Remove depracated packages and clean cache: 
sudo apt autoremove -y && sudo apt clean
Reboot: 
sudo reboot
See existence of overlayroot: 
df -h
check speed:
sudo dd if=/dev/random of=file bs=1M count=3076 status=progress;sync;
sudo dd if=/dev/zero of=file bs=1M count=3076 status=progress;sync;
df -h
rm file
